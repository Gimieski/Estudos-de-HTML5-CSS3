Links

Links são hiperlinks(endereço), que fazem hiper-ligações no www(word wibe web) por meio de endereços(URLS).

Links Internos

São aqueles links dentro da pagina,ou seja, aqueles feitos dentro do arquivo.Ex:

Link interno para o projeto strata.

Links externos

São links de outros lugares.Ex:

Link para a RiotGames

Atributo target e rel

Target

Define onde exibir o URL vinculado, como o nome de um contexto de navegação (uma guia, janela ou iframe). As seguintes palavras-chave têm significados especiais para onde carregar o URL

Valores

_self:Faz com que continue na mesma página o link. _blanck:Faz com que abra em outra aba a pagina carregada.

Rel

Rel é utilizado para falar qual é a categoria do arquivo que estamos linkando, por isso passamos o valor stylesheet para o atributo rel quando estamos linkando uma forma de estilo no atributo href

Exemplos:

Devolper.com

Devolper.com

Link para Download

Outra coisa que aparece bastante em sites são os links para efetuar download de algum material em PDF, ou de um arquivo ZIP qualquer. A partir da versão HTML5, as âncoras receberam atributos especiais para isso. Basta fazer o link diretamente para o arquivo que se deseja efetuar o download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado. Vamos ver:

Arquivo pdf

Nesse pdf exprime as informações sobre dowload e tipos.Mas já adiantando, type:é o tipo de arquivo.download:coloca o arquivo que você deseja baixar.

Sobre o Icone

Bom, sobre o icone, para colocarmos ele como icone de favorito,devemos ter um icone instalado,obviamente.Depois introduzimos ele no cabeçalho, com a tag "link". Com isso identificamos ele como link com o atributo "rel" e colocamos o icone com o atributo "href".
